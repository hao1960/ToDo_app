# ToDo应用发布前检查清单

## 打包前准备

1. [ ] 运行 clean_before_packaging.bat 清理任务数据和设置文件
2. [ ] 在干净的环境中编译Release版本
3. [ ] 确保图标文件(todolist.ico)存在于正确位置
4. [ ] 准备好字体文件和其他资源

## 文件检查

1. [ ] 确认没有包含本地任务数据文件(tasks_*.json)
2. [ ] 确认没有包含用户设置文件(*.ini)
3. [ ] 确认所有必要的DLL和依赖项已准备好
4. [ ] 确认README.md和LICENSE文件已更新

## 构建安装程序

1. [ ] 更新InnoSetup脚本中的版本号(当前版本: 2.0.0)
2. [ ] 编译InnoSetup脚本生成安装程序
3. [ ] 测试安装程序是否正常工作

## 测试验证

1. [ ] 在干净的系统上安装应用
2. [ ] 验证首次运行时是否正确初始化默认分类
3. [ ] 验证任务添加、编辑、标记完成功能
4. [ ] 验证应用其他功能是否正常工作

## 发布

1. [ ] 准备Release说明
2. [ ] 上传安装程序到GitHub Release
3. [ ] 更新在线文档
